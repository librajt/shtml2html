##接口说明：

每个接口返回数据类型为 json ，数据结构大概为：

{
    status: 1,  // 接口返回状态码，数字，1成功，0失败
    data: object  // 接口返回的数据，json对象或数组。若无则为null
}



##接口列表：

1. 首页
a. 加载更多
    参数：页数
    返回：分页数据


2. 分类
无


3. 分类筛选
a. 排序
    参数：分类id， 排序规则id
    返回：分类下按排序规则排序的商品列表
b. 显示更多
    参数：页数
    返回：分页数据


4. 商户介绍
无


5. 购买须知
无


6. 我的定单
a. 加载更多
    参数：页数
    返回：分页数据


7. 详情
a. 商品加入购物车
    参数：商品id、颜色、尺寸、数量
    返回：成功/失败状态码
b. 商品移出购物车
    参数：购物车商品id
    返回：成功/失败状态码
c. 购物车列表
    参数：无
    返回：购物车商品列表


8. 确认定单
无


9. 支付成功
无
